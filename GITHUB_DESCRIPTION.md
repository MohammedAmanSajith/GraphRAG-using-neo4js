### Long Description (for README or detailed overview)
Graph RAG System with Neo4j and Google Gemini

A comprehensive Graph-based Retrieval Augmented Generation (RAG) system that intelligently converts PDF documents into structured knowledge graphs. Built with LangChain, Neo4j, and Google Gemini, this system enables natural language querying of your document knowledge base.

**Key Features:**
- 📄 Automatic PDF to Knowledge Graph conversion
- 🧠 LLM-powered entity and relationship extraction
- 💬 Natural language chatbot interface
- 🔍 Intelligent Cypher query generation
- 📊 Schema-aware RAG with few-shot learning
- ⚡ Batch processing with configurable chunking

**Tech Stack:**
- LangChain - LLM orchestrationa framework
- Neo4j Aura - Graph database
- Google Gemini - Large Language Model
- Python 3.8+
